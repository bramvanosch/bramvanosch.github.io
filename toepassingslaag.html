Toepassingenlaag

Dit is de bovenste laag van het 3 lagenmodel. Jij als gebruiker bent maakt dus direct gebruik van deze laag. Hier bevinden zich de applicaties en toepassingen van je apperaat, beide zijn software. Voorbeelden van deze apps zijn: tiktok, safari en de spellen die je speelt op je computer/telefoon.

Applicaties

Applicaties zijn de toepassingen waar jij als gebruiker bij kan. Het is dus de software waar jij als gebruiker interactie mee kan hebben. Ze zijn dus vaak ook gemaakt om makkelijk te gebruiken en te downloaden te zijn.

API'S

Een API (application program interface) staan op je computer om het mogenlijk te maken dat applicaties op verschillende apparaten kunnen werken zonder speciale code nodig te hebben voor elk individueel apparaat. De API bevind zich tussen de toepassing en fysieke laag. Wil je meer weten over API'S? Klik dan hier!
